## Filosofía

Comply360 reemplaza un asesor laboral o un estudio jurídico chico (S/3K-S/15K/mes
en Lima). El pricing está anclado al **costo de un abogado por consulta** y a la
**multa SUNAFIL evitada**, no al costo de servir.

**Posicionamiento**: "Buk hace planillas. Comply360 hace que SUNAFIL nunca te
visite."

## Trial: 30 días + freemium post-trial

```
Días 1-30:   Trial PRO completo (todas las features de PRO)
Día 31+:     Auto-downgrade a FREE
             - Datos PRESERVADOS (NO se borran trabajadores, contratos, alertas)
             - Features pro bloqueadas con upsell modal contextual
             - Banner permanente: "Activa un plan para volver a PRO"
```

**Por qué freemium > paywall hard**:
- Paywall hard → cliente abandona, churn 100%
- Freemium → cliente sigue logueando para ver sus datos. Cada login = oportunidad
  de upsell. Notion/Slack/Figma usan exactamente este modelo.

## Decisiones explícitamente RECHAZADAS

| Decisión | Razón |
|----------|-------|
| ❌ White-label | Comply360 mantiene brand equity. No queremos diluir la marca. |
| ❌ Cobro por boleta | Política: boletas ilimitadas para todos. Marketing claim. |
| ❌ Founding Members lifetime puro | Lock-in muy pesado financieramente. 2 años es suficiente. |
| ❌ Trial de 14 días | Producto vertical-deep necesita 30 días para que cliente vea valor. |
| ❌ Paywall hard post-trial | Freemium retiene 30-40% más que paywall hard. |
